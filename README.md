# Vampire Survivor Clone

## Introduction
This project is a clone of the popular game "Vampire Survivor." It is developed using Pygame library. The goal is to survive waves of enemies and score as high as possible.

## Features
Multiple enemy types with unique behaviors
Player character with different abilities
Power-ups and upgrades
Scoring system
Increasing difficulty levels

## Installation & Usage
To run this project, you need to have Python and pygame library installed on your machine.

Use the arrow keys to move your character and avoid enemies. Collect power-ups to enhance your abilities and survive as long as possible.
