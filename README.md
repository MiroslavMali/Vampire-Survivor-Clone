# Vampire Survivor Clone

![VampireSurvivorClone-ezgif com-optimize](https://github.com/MiroslavMali/Vampire-Survivor-Clone/assets/68731924/83bee461-d6d3-42a5-9b9c-ba2324e57c28)


## Introduction
One of my most complex projects so far that I'm really proud of. This project is a clone of the popular game "Vampire Survivor." It is developed using Pygame library. The goal is to survive waves of enemies and score as high as possible.

## Key Features

- **Dynamic Player Mechanics**:
  - Smooth movement and detailed animations.
  - Health and experience points system for leveling up.

- **Engaging Enemy Interactions**:
  - AI-driven enemy behavior.
  - Enemies drop items like health potions and experience gems.

- **User Interface and Game States**:
  - Visual health and experience bars.
  - Real-time scoreboard.
  - Main menu and in-game pause menu.

- **Game State Management**:
  - Efficient transitions between playing, paused, and game over states.

- **Level and Environment Design**:
  - Automatic loading and updating of game levels.
  - Smooth camera system that follows the player.

- **Combat System**:
  - Interactive slash attacks with animations.
  - Experience gain from defeating enemies.

- **Settings and Customization**:
  - Configurable game settings for a personalized experience.

- **High-Quality Animations**:
  - Detailed sprite animations for enhanced visual appeal.
